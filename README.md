# TO-MY-SPECIAL

A simple static HTML website made as a romantic proposal page.

## About

This project is a single-page website built with plain HTML, CSS, and JavaScript. It includes animated stars and petals, a photo section, a custom music player, and interactive proposal buttons.

## Files

- `blossom-proposal.html` - main website file
- `images/` - photo assets used by the page
- `Yun Hi Tanu Weds Manu 128 Kbps.mp3` - background music file

## How To Open

No installation, build step, or Vercel deployment is needed.

1. Download or clone this repository.
2. Open `blossom-proposal.html` directly in any modern browser.

## Tech Used

- HTML5
- CSS3 animations
- Vanilla JavaScript

## Note

This is a personal static website. All assets are included in the repository so it can run locally from the HTML file.
